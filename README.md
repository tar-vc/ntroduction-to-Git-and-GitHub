# Simple Interest Calculator

A simple interest calculator script written in Bash.

## Description

This project contains a simple interest calculator that takes principal amount, annual rate of interest, and time period in years as input and calculates the simple interest.

## Formula

Simple Interest = P x T x R / 100

Where:
- **P** = Principal Amount
- **T** = Time Period (in years)
- **R** = Annual Rate of Interest

## Usage

```bash
bash simple-interest.sh
```
